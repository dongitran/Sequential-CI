Generate data using JavaScript code

Using JavaScript code can generate data to be used in other processes or create constant values.

With configuration: id = 1, phone number = '09' + Math.floor(Math.random()*99999999).toString().padStart(8, '0'), and accessKey = "494520340234509234"

Generate data with parameters

Can replace parameter strings with existing parameters, using the format {parameters['name-of-parameter']}.

Then parameters have id = 1 and idTmp = 10001.
